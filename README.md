# libraries-JSON
# План выполнения задания: Сравнение библиотек JSON для C++

##  Основные критерии оценки библиотек для работы с JSON

1. **Скорость чтения**
2. **Скорость записи**
3. **Удобство API**
4. **Использование оперативной памяти**
5. **Размер библиотеки**
6. **Поддержка потоковой обработки (стриминг)**
